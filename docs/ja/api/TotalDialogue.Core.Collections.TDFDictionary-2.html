    Class TDFDictionary<TKey, TValue>

    このクラスは、TDFKeyValuePairを継承し、辞書としての機能を提供します。 TDFKeyValuePairのリストを内部に持ち、キーと値のペアを管理します。 このクラスは、System標準のDictionaryと同じように動作しますが、シリアライズ可能です。 TDFListのスレッドセーフ/ロック機能とイベント機能を持ちます。 TDFListの要素数を取得するには、ListCountプロパティを使用してください。
